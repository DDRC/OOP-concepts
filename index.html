Conceptos de programacion orientada a objetos

Asociacion

Es una relacion de dos o más objetos, donde cada objeto es independiente.

El siguiente ejemplo aclarara la Asociacion

            function Person(name, surname) {
                this.name = name;
                this.surname = surname;
                this.parent = null;
                }
                var johnSmith = new Person("John", "Smith");
                var fredSmith = new Person("Fred", "Smith");
                fredSmith.parent = johnSmith;

Agregacion

La Agregacion es una forma especial de asociacion, donde un objeto tiene mayor importancia que otro , usualmente el objeto con mayor importancia determina el tipo de propiedad que un objeto tiene con otro.

El siguiente ejemplo demostrara el funcinamiento

            var company = {
                name: "ACME Inc.",
                employees: []
               };
               var johnSmith = new Person("John", "Smith");
               var marioRossi = new Person("Mario", "Rossi");
               company.employees.push(johnSmith);
               company.employees.push(marioRossi);

Composicion

Es un tipo fuerte de agregacion, donde cada objeto es dependiente.

El siguiente ejemplo demostrara el funcinamiento

            var person = {name: "John",
 surname: "Smith",
 address: {
 street: "123 Duncannon Street",
 city: "London",
 country: "United Kingdom"
 }};

Encapsulacion

Es la representacion de un objeto(incluidos propiedades y metodos) en una sola entidad.

El siguiente ejemplo demostrara el funcinamiento

            var company = {
                name: "ACME Inc.",
                employees: [],
                sortEmployeesByName: function() {...}
               };
               * Aqui usamos el objeto company que ha sido encapsulado.
            company.employees = "this is a joke!";
            company.sortEmployeesByName();

Herencia

Esta permite que nuevos objetos obtengan propiedades y metodos ya existentes en otros objetos, usualmente se crean objetos especializados a partir de objetos con caracteristicas mas generales.

El siguiente ejemplo demostrara el funcinamiento

            function Person() {
                this.name = "";
                this.surname = "";
            }
            function Programmer() {
                this.knownLanguage = "";
            }
            * Aqui se da la herencia al objeto programador que es una persona tambien.
               Programmer.prototype = new Person();
               
               var programmer = new Programmer();

Polimorfismo

En algunos casos el polimorfismo ayuda a reducir el codigo, pero ¿Que es el polimorfismo?. El polimorfismo es realizar una misma accion, que puede variar en cada objeto.

El siguiente ejemplo demostrara el funcinamiento

            public int Sum(int x, int y) {
                return Sum(x, y, 0);
               }
               public int Sum(int x, int y, int z) {
                return x+ y + z;
               }
               
        * En este ejemplo ambas funciones tienen un mismo nombre, pero 
        su funcion es distinta.

niveles de privacidad usando cierres

Es necesario tener claro algunos conceptos como el cierre que es la accesibilidad que se tiene cada variable de cada funcion al estar esta activa.

el alcance es la capacidad de que tiene un metodo anidado de acceder a otro externo pero no viceversa

el ES6 implementa los weakmaps, que son colecciones de pares de valores, que son una clave y un valor, las claves necesariamente deben ser objetos

El siguiente ejemplo ilustra los weakmaps

            var myMap = new WeakMap();
            var johnSmith = {name: "John", surname: "Smith"};
            var marioRossi = {name: "Mario", surname: "Rossi"};
            myMap.set(johnSmith, "This is John");
            myMap.set(marioRossi, "This is Mario");
            console.log(myMap.get(marioRossi));

los getters(adquiridor) son basicamente funciones que devuelven un valor que es asignado a una varible dentro de una clase

los setter(definidores) son funciones que definen el valor de una propiedad dentro de una clase. Lo cual permite actualizar los valores de las propiedades en las instancias creadas.

el siguiente ejemplo muestra el uso de un getter y un setter

            var person = {
                name: "John",
                surname: "Smith",
                get fullName() { return this.name + " " + this.surname; },
            set fullName(value) {
                var parts = value.toString().split(" ");
                this.name = parts[0] || "";
                this.surname = parts[1] || "";
                },
                email: "john.smith@packtpub.com"
               };

la implementacion seria la siguiente: 

            console.log(person.fullName); //John Smith
            person.fullName = "Mario Rossi";
            console.log(person.name); //Mario
            console.log(person.surname); //Rossi
            console.log(person.fullName); //Mario Rossi
